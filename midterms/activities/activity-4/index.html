Activity 4: Student Enrollment Assessment System

Christopher Balana | BSIT 2-5


This activity focuses on creating a student enrollment assessment system in Java, applying conditional logic, user input handling, and formula-based evaluation. It demonstrates how the Scanner class collects data, how nested if statements determine admission results, and how weighted scores combine exam and interview performance to simulate a real-world decision process. The exercise reinforces the importance of structuring conditions properly and applying formulas accurately to produce reliable outcomes.


For me, the task felt clear and methodical. The step-by-step rules made it easy to translate the requirements into code, and the logic flowed naturally once the conditions were set. It was more about careful organization than difficulty, making the activity a smooth and rewarding experience.
