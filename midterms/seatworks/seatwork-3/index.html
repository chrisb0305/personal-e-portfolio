In this activity, we utilized methods in Java programming. Methods are blocks of code that can be reused by calling them in the main method. We also used the Scanner class for input and output. The program’s process was to classify the grade entered by the student as Excellent, Very Good, Good, or Failed.


The activity was easy for me since I had already grasped the concept of methods—similar to functions in Python programming—as well as the Scanner class. Overall, it was a very fun and worthwhile task.


StudentGradeEvaluator.java 
import java.util.Scanner;

    public class StudentGradeEvaluator {
    
      // Method to validate grade input
      public static double getGrade(Scanner input, String gradeType) {
        double grade = -1;
        while (grade < 0 || grade > 100) {
          System.out.print(gradeType + " Grade: ");
          if (input.hasNextDouble()) {
            grade = input.nextDouble();
            if (grade < 0 || grade > 100) {
              System.out.println("Invalid grade input.");
            }
          } else {
            System.out.println("Invalid grade input.");
            input.next(); // clears invalid input
          }
        }
        return grade;
      }

      // Method to compute average
      public static double computeAve(double pre, double mid, double fin) {
        return (pre + mid + fin) / 3;
      }

      // Method to determine remark
      public static String gradeMark(double average) {
        if (average >= 90 && average <= 100) {
          return "Excellent";
        } else if (average >= 80 && average <= 89) {
          return "Very Good";
        } else if (average >= 75 && average <= 79) {
          return "Good";
        } else {
          return "Failed";
        }
      }

      // Main method
      public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        // Get student name
        System.out.print("Student Name: ");
        String studentName = input.nextLine();

        // Get and validate grades
        double prelimGrade = getGrade(input, "Prelim");
        double midtermGrade = getGrade(input, "Midterm");
        double finalGrade = getGrade(input, "Final");

        // Compute average
        double average = computeAve(prelimGrade, midtermGrade, finalGrade);

        // Output
        System.out.println("\nStudent Name: " + studentName);
        System.out.printf("Average: %.2f\n", average);
        System.out.println("Remark: " + gradeMark(average));

        input.close();
      }
    }
