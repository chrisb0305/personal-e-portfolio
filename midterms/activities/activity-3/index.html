Activity 3: Basic ATM System

Christopher Balana | BSIT 2-5


This activity centers on building a basic ATM system in Java, applying key programming concepts such as user input with the Scanner class, decision structures through switch and if–else statements, and repetition using a do-while loop. It demonstrates how logic controls program flow, validates conditions like withdrawal limits, and maintains an updated balance, reinforcing the importance of combining operators, control structures, and loops to simulate real-world applications.


For me, the task felt engaging rather than difficult. Designing the ATM system was straightforward once the rules were clear, and it gave me a sense of how programming can model everyday processes. It was satisfying to see the program respond interactively, making the activity both practical and enjoyable.
