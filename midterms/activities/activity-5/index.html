Activity 5: Personal Expense Tracker

Christopher Balana | BSIT 2-5


This activity is about designing an expense tracker in Java, where different methods handle tasks such as displaying the program title, calculating expenses, checking budget status, and showing results. It emphasizes modular programming, proper use of return types, parameters, and method calls, while also encouraging enhancements like user input, extra categories, or personalized messages. The exercise demonstrates how Java’s flexibility allows developers to combine various aspects—methods, conditionals, and input handling—to build practical and organized applications.


For me, the task felt very easy and gave a sense of freedom to explore different Java features. I could mix methods, add enhancements, and experiment with how the program responds, which made it less restrictive and more creative. It was enjoyable to see how simple concepts could be expanded into a functional system, making the activity both effortless and liberating.
