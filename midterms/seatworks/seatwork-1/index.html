Christopher Balana | BSIT 2-5


I don’t have a copy of this seatwork because it was done during face-to-face classes. But as far as I remember, it was about increment and decrement operators in Java.


It was an easy task because I was able to answer correctly and I got a perfect score for it. Here is an example of Incerement and Decrement operations in Java.

Increment_Decrement_Operators.java 
int a = 20, b = 15, c = 5, d = 10;

// a uses 20 then becomes 21, b uses 15 then becomes 16
a++ + b++;    // 20 + 15 = 35

// b uses 16 then becomes 17, c decreases to 4 and is used
b++ - --c;    // 16 - 4 = 12

// a increases to 22 and is used, c decreases to 3 and is used
++a + --c;    // 22 + 3 = 25

// b increases to 18 and is used, a increases to 23 and is used
++b + ++a;    // 18 + 23 = 41

// d uses 10 then becomes 11, a increases to 24 and is used
d++ + ++a;    // 10 + 24 = 34
